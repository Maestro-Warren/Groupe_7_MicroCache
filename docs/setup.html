Installation complète

Guide d'installation et de configuration

Toutes les étapes nécessaires pour lancer MicroCache localement, installer les dépendances et configurer les composants Rust et Node.js.

Prérequis

  • Rust 1.70+ avec Cargo.
  • Node.js 18+ et npm.
  • SQLite accessible via Node.js experimental SQLite.
  • Internet pour charger Tailwind et Mermaid dans la doc.

Installation pas à pas

1. Cloner le dépôt

git clone https://github.com/Maestro-Warren/Groupe_7_MicroCache

2. Installer les dépendances Node.js

cd microcache-demo/api-server
npm install

3. Compiler le serveur Rust

cd ../..
cargo build --release

Configuration recommandée

Utilisez une configuration de port standard pour éviter les conflits : MicroCache sur 6379 et API sur 3000. Assurez-vous que les deux services peuvent communiquer localement.

  • MicroCache écoute sur 127.0.0.1:6379.
  • API backend sur http://localhost:3000.
  • Dashboard accessible sur http://localhost:3000.

Démarrage rapide

Lancer MicroCache :

cargo run

Lancer l’API :

cd microcache-demo/api-server
npm start

Ouvrir le dashboard :

http://localhost:3000

Conseils de déploiement

Pour un déploiement réel, séparez le backend de l’API et le serveur MicroCache sur différents conteneurs ou VM. Utilisez un load balancer pour répartir les requêtes API et surveillez l’utilisation mémoire du cache.
